▸ 核心公式速查
发动机转矩（4次多项式）: Tq = A + B·(n/1000) + C·(n/1000)² + D·(n/1000)³ + E·(n/1000)⁴
驱动力: Ft = Tq·ig·i₀·ηt / r
车速: ua = 0.377·r·n / (ig·i₀) [km/h]
行驶阻力: F = f·ma·g + CDA·ua² / 21.15 [平路, N]
旋转质量系数: δ = 1 + (If·ig²·i₀² + Iw1 + Iw2) / (ma·r²)
加速度: j = (Ft – Fr) / (δ·ma) [m/s²]
最大爬坡度: sinα_max ≈ (Ft1_max – f·ma·g) / (ma·g)
后轴附着率: φ = Ft / N₂；N₂ = ma·g·[a/L·cosα + hg/L·(sinα – f·cosα)]
▸ 检查要点
#检查项目易错点
①驱动力–阻力平衡图各档曲线从n_min到n_max；阻力曲线是否包含空气阻力
②最高车速应是最高档Ft=Fr交点，不是发动机最高转速对应车速
③最大爬坡度用1档最大Ft；需减去滚动阻力；爬坡度%=tan(α)×100
④附着率后轴法向载荷要考虑坡道和惯性影响；检查是否<路面φ
⑤1/j曲线δ计算中ig和i₀的平方；各档曲线对应速度范围正确
⑥v-t曲线梯形积分Δt=Δv/j；换挡判断（n_max时升档）；总时间
